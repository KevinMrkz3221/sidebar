## üöÄ Installation (Developer Mode)

Since this extension is in active development, you can install it manually:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/KevinMrkz3221/sidebar.git
    ```
2.  **Open Chrome Extensions**:
    Navigate to `chrome://extensions/` in your browser.
3.  **Enable Developer Mode**:
    Toggle the switch in the upper right corner to **ON**.
4.  **Load Unpacked**:
    Click the **"Load unpacked"** button and select the project folder.
